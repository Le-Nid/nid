# Recherches sauvegardées

Sauvegardez vos filtres de recherche Gmail favoris pour les réutiliser en un clic.

---

## Créer une recherche sauvegardée

1. Ouvrez la page **Recherches sauvegardées** dans la sidebar
2. Cliquez sur **Nouvelle recherche**
3. Renseignez :
    - **Nom** : un libellé descriptif (ex. : "Factures récentes")
    - **Requête** : la requête Gmail à sauvegarder (ex. : `from:amazon.fr subject:facture newer_than:6m`)
    - **Icône** : choisissez une icône pour identifier visuellement la recherche
    - **Couleur** : choisissez une couleur

> 📸 *Capture d'écran suggérée : formulaire de création d'une recherche sauvegardée avec les champs nom, requête, icône et couleur*

---

## Utiliser une recherche sauvegardée

Cliquez sur une recherche sauvegardée pour exécuter la requête associée. Les résultats s'affichent dans la page Mes mails.

> 📸 *Capture d'écran suggérée : liste des recherches sauvegardées avec les icônes colorées et les noms*

---

## Réorganiser

Glissez-déposez les recherches sauvegardées pour les réordonner selon vos préférences.

---

## Modifier et supprimer

- Cliquez sur l'icône d'édition pour modifier une recherche (nom, requête, icône, couleur)
- Cliquez sur l'icône de suppression pour la supprimer

---

## Exemples de recherches utiles

| Nom | Requête | Usage |
|---|---|---|
| Factures récentes | `subject:(facture OR invoice) newer_than:6m` | Retrouver rapidement les factures |
| Gros mails | `larger:10M` | Identifier les mails volumineux |
| Non lus anciens | `is:unread older_than:30d` | Nettoyer les mails non lus oubliés |
| PJ importantes | `has:attachment from:(banque OR assurance)` | Pièces jointes de services importants |
| Promotions | `category:promotions older_than:7d` | Nettoyer les promos périmées |
