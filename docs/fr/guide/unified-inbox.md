# Boîte unifiée

Si vous avez connecté plusieurs comptes Gmail, la **boîte unifiée** vous permet de consulter tous vos mails en une seule vue.

---

## Vue d'ensemble

La page **Boîte unifiée** fusionne les mails de tous vos comptes Gmail connectés dans une **timeline unique**, triée par date.

Chaque mail est identifié par un **tag coloré** indiquant le compte Gmail d'origine.

> 📸 *Capture d'écran suggérée : page Boîte unifiée avec des mails provenant de différents comptes, chacun identifié par un tag de couleur différente*

---

## Comment ça marche

1. L'application interroge **simultanément** tous vos comptes Gmail
2. Les résultats sont fusionnés et triés par date décroissante
3. Si un compte est temporairement indisponible, les mails des autres comptes sont quand même affichés

---

## Utilisation

- La barre de recherche fonctionne sur **tous les comptes** simultanément
- Cliquez sur un mail pour l'ouvrir dans le lecteur intégré
- Le tag coloré vous indique à quel compte appartient chaque mail

---

## Limitations

- La boîte unifiée est en **lecture seule** : les actions en masse ne sont pas disponibles depuis cette vue
- Pour agir sur des mails (supprimer, archiver, etc.), naviguez vers **Mes mails** et sélectionnez le compte concerné
