# Saved Searches

Save your favorite Gmail search filters to reuse them with a single click.

---

## Create a Saved Search

1. Open the **Saved Searches** page in the sidebar
2. Click **New search**
3. Fill in:
    - **Name**: a descriptive label (e.g., "Recent invoices")
    - **Query**: the Gmail query to save (e.g., `from:amazon.com subject:invoice newer_than:6m`)
    - **Icon**: choose an icon to visually identify the search
    - **Color**: choose a color

> 📸 *Suggested screenshot: saved search creation form with name, query, icon, and color fields*

---

## Use a Saved Search

Click on a saved search to execute its associated query. Results are displayed on the My Emails page.

> 📸 *Suggested screenshot: list of saved searches with colored icons and names*

---

## Reorder

Drag and drop saved searches to reorder them according to your preferences.

---

## Edit and Delete

- Click the edit icon to modify a search (name, query, icon, color)
- Click the delete icon to remove it

---

## Useful Search Examples

| Name | Query | Usage |
|---|---|---|
| Recent invoices | `subject:(facture OR invoice) newer_than:6m` | Quickly find invoices |
| Large emails | `larger:10M` | Identify large emails |
| Old unread | `is:unread older_than:30d` | Clean up forgotten unread emails |
| Important attachments | `has:attachment from:(bank OR insurance)` | Attachments from important services |
| Promotions | `category:promotions older_than:7d` | Clean up expired promotions |
